# Spoorthika Reddy Legala Portfolio

A professional portfolio website showcasing my background, technical skills, featured projects, resume, and contact details.

## Features

- Clean single-page portfolio layout
- Dedicated HTML resume page
- Project showcase with repository links
- Downloadable resume PDF
- Contact links for email, GitHub, and LinkedIn

## Built With

- HTML
- CSS
- JavaScript

## Files

- `index.html` - portfolio homepage
- `style.css` - portfolio styling
- `script.js` - project cards and interactions
- `resume.html` - resume page
- `resume.css` - resume styling
- `favicon.svg` - browser tab icon
- `Spoorthika_Reddy_Legala_Resume.pdf` - resume PDF

## How To View

1. Open `index.html` in a browser for the portfolio.
2. Open `resume.html` in a browser for the resume page.
3. Use the browser print option on the resume page to save it as PDF if needed.
